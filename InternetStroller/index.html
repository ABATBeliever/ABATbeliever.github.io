Welcome to InternetStroller!

InternetStrollerは非常に軽量な、オープンソースのウェブブラウザです
InternetStroller is a lightweight, open source web browser

  • インストール不要。環境を汚さず、USBなどに入れて持ち運べます。
  • 履歴やサインインデータ等は同じディレクトリに保存。ネットを散歩しましょう。
  • WebView2による、堅牢で快適なブラウジング体験。
  • PNGやPDFのスクリーンショットも簡単に。
  • 他のブラウザへのURL送信機能。

  • No installation required.
  • Portable via USB.
  • Robust and comfortable browsing experience with WebView2.
  • Easy PNG and PDF screenshots.
  • Ability to send URLs to other browsers.
